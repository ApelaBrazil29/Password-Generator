# 🔐 Random Password Generator

A modern and secure **Password Generator Web App** built using HTML, CSS, and JavaScript. It generates strong random passwords and allows users to copy them instantly for secure usage.

---

## 🚀 Live Demo

🔗 https://apelabrazil29.github.io/Password-Generator/

---

## 📌 Features

* 🔐 Generate strong random passwords
* 🔢 Custom password length (user input)
* 📋 One-click copy to clipboard
* ⚡ Instant password generation
* ⚠️ Input validation (length limits)
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Clipboard API

## ⚙️ How It Works

1. Enter desired password length
2. Click **Generate Passwords**
3. Two random passwords are generated
4. Click on any password to copy it
5. Confirmation message appears

---

## ⚠️ Validations

* ❌ Length cannot be less than 1
* ❌ Length cannot exceed 20
* ✅ Only valid input is accepted
